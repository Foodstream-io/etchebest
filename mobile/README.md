# etchebest
EIP

## Getting Started

To start the development server:
```bash
npx expo start
```

## Running on Device/Simulator

To run on Android:
```bash
npx expo run:android
```

To run on iOS:
```bash
npx expo run:ios
```

## Building the App

To build for production:
```bash
npx expo build:android
npx expo build:ios
```

## Re-running the Project

If you need to restart the development server:
```bash
npx expo start -c
```
